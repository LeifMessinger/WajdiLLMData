Creating a simple single-page website for your app "WhereIs?" involves HTML for the structure and CSS for styling. Since you've requested to include the CSS inside the HTML using <style> tags, I'll provide a basic example that meets your requirements. This example will include a header, body, and footer with some filler information about your app.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>WhereIs? - Find Your Lost Items</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
            background-color: #f0f0f0;
        }
        header {
            background-color: #333;
            color: white;
            padding: 10px 0;
            text-align: center;
        }
        .container {
            margin: 20px auto;
            max-width: 800px;
            padding: 20px;
            background-color: white;
            box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
        }
        footer {
            background-color: #333;
            color: white;
            text-align: center;
            padding: 10px 0;
            position: fixed;
            bottom: 0;
            width: 100%;
        }
    </style>
</head>
<body>
    <header>
        <h1>WhereIs?</h1>
        <p>Find Your Lost Items</p>
    </header>
    <div class="container">
        <h2>About WhereIs?</h2>
        <p>WhereIs? is a revolutionary app designed to help you find your lost items. With just a few taps, you can locate your misplaced belongings, whether they're in your house, your car, or even in a public place. Our advanced technology uses GPS and Bluetooth to pinpoint your items with unmatched accuracy.</p>
        <h2>Features</h2>
        <ul>
            <li>GPS and Bluetooth tracking for precise location</li>
            <li>Real-time updates on your lost items</li>
            <li>Easy to use interface</li>
            <li>Secure and private</li>
        </ul>
        <h2>Download Now</h2>
        <p>Available on the App Store and Google Play.</p>
    </div>
    <footer>
        <p>&copy; 2023 WhereIs? All rights reserved.</p>
    </footer>
</body>
</html>

This HTML document starts with a <!DOCTYPE html> declaration, which is necessary for HTML5. The <head> section includes meta tags for character encoding and viewport settings, which are important for ensuring your website displays correctly on all devices. The <title> tag sets the title of the webpage, which appears in the browser's title bar or tab.

The <style> section within the <head> contains CSS rules for styling the page. The body styles apply to the entire page, setting the font, margin, padding, and background color. The header, .container, and footer styles define the appearance of these sections.

The <body> section contains the content of the webpage. It starts with a <header> that includes the app's name and a brief description. The main content is wrapped in a <div> with the class container, which includes sections about the app, its features, and a call to action to download the app. Finally, a <footer> provides copyright information.

This is a basic example to get you started. You can customize the content and styles further to better match your app's branding and functionality.
