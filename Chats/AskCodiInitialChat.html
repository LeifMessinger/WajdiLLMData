<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>WhereIs? App</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }
        header {
            background-color: #4CAF50;
            color: white;
            text-align: center;
            padding: 10px;
        }
        footer {
            background-color: #4CAF50;
            color: white;
            text-align: center;
            padding: 10px;
            position: fixed;
            bottom: 0;
            width: 100%;
        }
        .container {
            padding: 20px;
        }
    </style>
</head>
<body>
    <header>
        <h1>Welcome to WhereIs? App</h1>
    </header>

    <div class="container">
        <h2>About WhereIs? App</h2>
        <p>WhereIs? is a revolutionary app that helps you find things you lose quickly and easily. With its advanced tracking technology, you'll never lose your keys, wallet, or anything else again!</p>
    </div>

    <footer>
        <p>&copy; 2022 WhereIs? App</p>
